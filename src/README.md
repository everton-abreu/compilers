# Análise Léxica

Para executar a análise sintátic, use o seguinte comando:
`python3 __lexer__.py <arquivo de teste>`

Os arquivos de teste estão salvos no diretório `lexer/tests`

# Análise Sintática

Para executar a análise sintátic, use o seguinte comando:
`python3 __syntatic__.py <arquivo de teste>`

Os arquivos de teste estão salvos no diretório `syntatic/tests`
